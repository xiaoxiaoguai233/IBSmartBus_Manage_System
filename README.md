# IB智慧公交管理系统
该项目是本人在大二实训时进行的入门训练，基于java中的SSM框架对数据库进行简单的增删改查的操作。

# 开发环境
IDE： Eclipse 
JDK： 1.8
Spring：4.0
SpringMVC：4.1
Mybatis：3.4.0
MySQL： 5.6
Tomcat： 7.0
Jquery： 3.7.0
BootStrap： 4.0

# 技术
基于JavaWeb中的主流运用框架SSM（Spring+SpringMVC+Mybatis），以及前端使用的Jquery + BootStrap4 的框架搭建。该系统主要对于SSM框架的基本入门，以及对数据库的简单的设计，操作（增删改查）。现已开源，借此弘扬互联网精神，共勉。
